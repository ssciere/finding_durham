About This Site



Hi, I'm Steve Sciere. I made this website to learn more about Web Development as I transition to a career in that field. I also wanted to make something to give back to the city of Durham, where I have lived for many years. I hope you'll consider trying out the businesses you discover here!

If you'd like to learn more about my development skills, you can do so at stevesciere.tech.

Technologies



I built this site using Python and Django as a web framework. A mathematical formula compares user preferences with data Django pulls from an SQLite database to find a business the user might like. I plan to add additional pieces to strengthen my Python and Django skills moving forward.
